# MMA Decoded Dashboard

Public tracking dashboard for [MMA Decoded](https://mmadecoded.com) — a quantitative research project investigating inefficiencies in MMA betting markets.

## What's on the dashboard

- **Track Record** — Model Brier score vs closing line, CLV gap, line movement confirmation rate, ROI at multiple edge thresholds, calibration plot, yearly breakdown, and 500 most recent predictions with per-fight line movement
- **Ratings** — Top 100 UFC fighters ranked by Glicko-2 mu, computed from 200K+ fights across all MMA promotions
- **Methodology** — How the model works: 21 curated delta features, Ridge-regularized logistic regression, walk-forward backtest with expanding windows
- **About** — Project overview and links

## Stack

- [Next.js](https://nextjs.org) (static export)
- [Tailwind CSS](https://tailwindcss.com) v4
- [Recharts](https://recharts.org)
- Deployed on [Vercel](https://vercel.com)

## Data

The dashboard reads static JSON files from `public/data/` at build time. No server, no API.

| File | Contents |
|------|----------|
| `metrics.json` | Brier scores, CLV, line movement, calibration, ROI |
| `predictions.json` | 500 most recent fight predictions with outcomes |
| `ratings.json` | Top 100 UFC fighter Glicko-2 ratings |
| `yearly.json` | Per-year performance breakdown |

Data is generated by the model pipeline and exported via `export_data.py`.
